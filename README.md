## Testing

- unit tests
```
npm run test:unit
```
e2e tests
```
npm run test:e2e
```

- dev mode unit tests
-
```
npm run test:unit-watch
```
e2e tests
```
npm run test:e2e-dev
```

- run all tests

```
npm run test
```
